# adder

```text
Usage: adder [options] <salutation>

Boiler plate calculator app to add two numbers

Arguments:
  salutation           A mandatory salutation argument

Options:
  -V, --version        output the version number
  -v, --verbose        Verbose mode, try it
  -n, --name <string>  Your name (default: "there")
  -o, --one <number>   The first operand
  -t, --two <number>   The second operand
  -h, --help           display help for command
```
